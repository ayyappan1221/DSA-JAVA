# Queue Question Bank (Basic to Advanced)

A curated list of queue practice questions for learning concepts in progression.

## Basic Level

- Implement a queue using an array
- Implement a queue using a linked list
- Implement enqueue, dequeue, front, rear, isEmpty, and size operations
- Display the queue without altering its contents
- Implement a circular queue using an array
- Check whether a queue is full or empty
- Reverse a queue using a stack
- Generate binary numbers from 1 to n using a queue

## Intermediate Level

- Implement a queue using two stacks
- Implement a stack using two queues
- Print the first non-repeating character in a stream using a queue
- Implement a double-ended queue (deque)
- Perform level-order traversal of a tree using a queue concept
- Rotate a queue by k positions
- Implement a ticket booking simulation using a queue

## Advanced Level

- Implement a priority queue using a heap logic (concept-based)
- Simulate task scheduling using round-robin with queue
- Implement producer-consumer problem using queue logic
- Simulate message queue handling in a system
- Cache eviction simulation using queue (FIFO strategy)
- Implement sliding window maximum using a deque logic
- Find shortest path in an unweighted graph using queue logic (BFS)
