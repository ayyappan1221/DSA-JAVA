# Stack Question Bank (Basic to Advanced)

A curated list of stack practice questions to learn concepts step-by-step.
These are topic-based and not tied to any platform.

## Basic Level

- Implement a stack using an array
- Implement a stack using a linked list
- Implement push, pop, peek, isEmpty, and size operations
- Reverse a string using a stack
- Check whether a word is a palindrome using a stack
- Convert a decimal number to binary using a stack
- Remove the middle element of a stack without using extra space
- Display the stack without altering its contents

## Intermediate Level

- Check whether parentheses in an expression are balanced
- Convert an infix expression to postfix
- Convert a postfix expression to infix
- Evaluate a postfix expression
- Evaluate a prefix expression
- Sort a stack without using another stack
- Reverse a linked list using a stack
- Check for redundant brackets in an expression
- Find the minimum element in a stack without using extra space

## Advanced Level

- Implement two stacks using a single array
- Implement a stack that supports undo-redo operations
- Simulate browser forward and backward navigation using stacks
- Implement a stack using queues (two approaches)
- Detect cycle in function calls using a call-stack concept
- Find the next greater element using a stack logic
- Solve expression evaluation with operator precedence handling
